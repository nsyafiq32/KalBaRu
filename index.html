Kalkulator Bangun Ruang

Persegi

Rumus pada Persegi yaitu :

L = s x s
K = 4 x s

Dimana :

L : Luas

K : Keliling

s : Sisi


Input Nilai Sisi

Persegi Panjang

Rumus pada Persegi Panjang yaitu :

L = p x l
K = 2 x (p + l)

Dimana :

L : Luas

K : Keliling

p : Panjang

l : Lebar



Input Nilai Panjang


Input Nilai Lebar
